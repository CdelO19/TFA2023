Développement

Les hypothèses choisies, je peux déjà réfléchir aux différentes fonctions que pourrait avoir mon application. Je vais utiliser trois modules principaux :

  • Les filtres

    Comme premier module pour mon application, je souhaite mettre en place un système de filtre pour rechercher les informations d'aide.

    Justification

    Utiliser des filtres pourraient faciliter la recherche des informations des utilisateurs, cela évitera de faire une recherche trop longue mais également d'avoir les informations les plus adéquates possibles.

  • Les informations

    Mon deuxième module serait de séparer les informations en 2 catégories : les informations pour les victimes et les informations pour les personnes souhaitant aider une victime.

    Justification

    Le fait de séparer les informations en 2 catégories permet de faire un premier tri. Les personnes victimes n'auront pas forcément les mêmes informations ni les mêmes besoins qu'une personne témoin ou qui souhaite aider quelqu'un.

  • Discussion

    Le troisième module consiste à créer une discussion avec une personne qualifiée. Comme une discussion messenger ou zoom.

    Justification

    Proposer aux victimes de parler via message ou appel permet de les mettre à l'aise avec leur interlocuteur. En effet, certaines victimes préfèrent écrire, d'autres appeler. Si elle est obligée de faire l'un ou l'autre, cela peut mettre un frein à la communication.
